# My Web GIS Map - Lab 6 Final
 
## Description
This web map shows 3 cities across the US with population information.
It was built using Leaflet.js as part of GIS 5220/420 Lab 6.

## Features
- Interactive basemap with zoom and pan
- Coordinates, state, city and population information is contained within a geojson.
- Interactive pop-ups display city, state, population and notes about each city. 
- Scale bar showing metric and imperial units
 
## Technologies Used
- HTML5, CSS3, JavaScript (all in one index.html file)
- Leaflet.js 1.9.4
- OpenStreetMap tiles
